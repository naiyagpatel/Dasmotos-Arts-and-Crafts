# Dasmoto's Arts & Crafts

## In this project, a simple website is created for a fictional arts and crafts store using the Visual Studio Code Editor based on spec file. ##
___
### Technologies used for this project ###
- HTML (used semantic elements)
- CSS

### Setup information ###

1. Create folder and name it relevant to your project in your local drive
2. Download all images provided in my GitHub account and save in your project under images folder (you need to create images folder in your project)
3. Save content of HTML and CSS files and save them in the same folder with .html and .css extensions
4. Run the .html file in your preference browser
5. Once you run the website you can see it should look similar to spec file provided here
